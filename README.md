# cloud-computing-Project
repository untuk projek mata kuliah cloud computing and security kelas C Prodi Teknik Komputer

# Langkah untuk menjalankan sistem di Instance EC2
1. Install Dependensi Python dengan "pip3 install flask requests"
2. Konfigurasi CloudWatch Agent: Pastikan agen memiliki file config.json yang diarahkan untuk memantau /home/ec2-
3. beri izin eksekusi pada skrip bash dengan "chmod +x chaos.sh run_demo.sh"

# Cara menjalankan Demo
ketik perintah "./jalankan_demo.sh" di terminal EC2
